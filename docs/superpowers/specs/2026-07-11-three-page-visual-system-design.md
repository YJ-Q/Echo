# Margin 三页统一视觉系统设计

## 目标

统一首页、成长轨迹和留痕三张功能页的视觉语言。以当前代码中的首页为唯一母版，保留首页左侧对话区的布局与交互，只调整其右侧硫酸纸内容；成长轨迹和留痕页沿用同一笔记本框架、字体层级、间距、控件和动效。

## 母版与共享框架

- 使用现有笔记本外壳、书脊、左侧主导航页签和右上角绸带。
- 桌面主区域保持首页的 58% / 42% 双页比例及现有安全边距。
- 中文统一使用宋体体系；仅 `Margin` 英文标识使用 Garamond。
- 一级标题 24–26 px，二级标题 15–17 px，正文 12–14 px，辅助信息 8–10 px。
- 共享色值：主文字 `#34312c`、次文字 `#746d63`、纸张 `#f8f4ea`、深纸色 `#eee7d9`、纸边 `#d8cfbf`、强调色 `#9a7442`。
- 分隔线使用低透明度 1 px 细线；纸张投影沿用首页的轻微左下阴影。
- 控件以无边框图标、文字按钮和细下划线为主，不使用常规网页式填充按钮。
- 页面或纸张展开约 320 ms，文字出现约 420 ms；减少动态偏好下关闭非必要动效。

## 首页

- 左侧 `ReflectiveMargin` 完全保持现状：用户消息右对齐，Margin 消息左对齐，输入、语音和阅读逻辑不变。
- 右侧保留硫酸纸覆盖结构，信息顺序为“我看见的”“我注意到”“还可以继续看看”。
- 当对话中出现适合持续练习的方向时，显示轻量的成长线建议，并由用户确认后创建。
- 成长建议使用与成长轨迹页“本周小实验”相同的纸条组件。

## 成长轨迹

- 左页从上到下展示“我现在的理解”“我注意到”“可能担心”“想试试看”、本周小实验和真实情境记录。
- 真实情境记录按当周日期切换，一次显示一天；左右箭头切换周，长记录在记录区域内滚动。
- 右页展示一条成长路径，顶部为学习线主题；仅展示上一个已完成节点、当前节点和下一个未开始节点。
- 三个节点按上、中、下错落排列，以空心或实心圆点和直线连接；滚轮浏览前后节点，已完成节点可查看，未开始节点不可点击。
- 路径底部以独立纸条切换其他成长线。
- 成长页不展示印记或硬币，为路径与记录留出空间。
- 使用绘图纸纹理和少量蓝灰色辅助标记，但所有文字层级、控件与首页一致。

## 留痕

- 默认展示最近十条“最近留下”，按实际日期分组，不使用“今天、昨天、更早”。
- 用户主动保存的内容和成长记录直接进入留痕；AI 观察必须经用户确认。
- 辅助区域展示“慢慢形成”和“最近印记”，并提供印记全集入口。
- 印记只在留痕页出现。最近三枚采用圆形金属硬币：暖炭黑底、低亮度金色描边和图案、无文字，缩小并紧密重叠。
- 页面边缘索引可以保留，但必须小于左侧主导航页签，贴合纸张边缘且降低对比度。
- 使用档案棉纸纹理和暖灰棕色，不改变共享字体与控件体系。

## 三种纸张角色

- 首页硫酸纸：正在发生的思考，最轻、最透明。
- 成长轨迹绘图纸：正在推进的练习，允许少量蓝灰辅助线。
- 留痕档案纸：已经留下的经验，略暖、略有纤维感。

材质差异只承担页面角色辨识，不改变共同的框架、排版和交互语言。

## 验收标准

- 三页并排时，外壳、页签、绸带、双页比例和内容安全边距一致。
- 同级标题、正文、辅助文字和交互控件在三页中具有相同字号、字重和颜色。
- 首页左侧与当前代码一致，生成稿不得重新解释其消息布局。
- 成长页没有印记区域；硬币只出现在留痕页。
- 三页可以通过纸张材质区分，但不会被误认为三个不同产品。
