# Margin material assets

These textures are independent raster layers generated for the approved paper system. Layout, edges, opacity, and shadows remain in CSS so the pages can move and animate independently.

- `vellum-v1.jpg`: homepage learning-line overlay
- `drafting-v1.jpg`: Learning Track page
- `archive-cotton-v1.jpg`: Traces and confirmation papers
- `ledger-v1.jpg`: organization overlay
- `settings-note-v1.jpg`: compact settings paper

The source generations were produced with the built-in image generation tool using the uploaded Margin reference for material and color direction. The checked-in JPEGs are quality-84 optimized copies of the generated PNG sources.
